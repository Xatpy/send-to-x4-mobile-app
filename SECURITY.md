# Security Policy

## Supported Versions

Security fixes are applied to the latest commit on `main`.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report vulnerabilities privately by email:
- `hi@chapiware.com`

Include:
- A clear description of the issue and impact
- Steps to reproduce
- A proof-of-concept if available
- A suggested fix (optional)

You will receive an acknowledgment within 5 business days. After validation,
we will work on a fix and coordinate disclosure timing with you.

## Scope Notes

This app communicates with devices on local networks and may process URLs and
files provided by users. Reports related to local-network abuse, content
parsing, and file handling are especially welcome.
